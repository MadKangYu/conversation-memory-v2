# Conversation Memory V2 - 제품 개요서 (POD)

---

## AI 에이전트, 어제의 대화를 기억하게 하라

**Conversation Memory V2는 AI 에이전트가 대화의 맥락을 잊어버리는 문제를 해결하는 차세대 장기 기억 시스템입니다. 단 한 번의 설치로, 당신의 AI 에이전트는 모든 대화를 기억하고, 더 스마트하고, 더 일관된 파트너가 됩니다.**

---

### 지금, 당신의 AI 에이전트가 겪는 문제

혹시 AI와 대화하다가 답답했던 적 없으신가요? 대화가 조금만 길어져도 AI는 방금 전에 나눴던 중요한 이야기를 잊어버립니다. 우리는 같은 말을 반복해야 하고, 작업의 흐름은 끊임없이 단절됩니다. 이는 AI 에이전트가 가진 **'기억의 한계'** 때문입니다.

| 문제점 | 사용자의 고통 |
| :--- | :--- |
| **기억 상실** | "아까 말했잖아!"를 외치며 같은 설명을 반복해야 합니다. |
| **수동 컨텍스트 관리** | 중요한 정보를 잊지 않게 하려고 직접 요약 노트를 만들어줘야 합니다. |
| **비용 낭비** | 모든 대화 내용을 억지로 컨텍스트에 넣다 보니 불필요한 API 비용이 발생합니다. |

### 우리의 해결책: 팩토리 드루이드 (Factory Druid)

Conversation Memory V2는 이 모든 문제를 **'팩토리 드루이드'** 라는 혁신적인 패턴으로 해결합니다. 사용자는 복잡한 설정 없이, 단 한 번의 설치만으로 이 모든 기능을 누릴 수 있습니다.

> **"설치하고, 잊어버리세요. 나머지는 저희가 알아서 합니다."**

이 시스템은 백그라운드에서 조용히 작동하며, AI 에이전트의 모든 상호작용을 자동으로 캡처하고 지능적으로 압축합니다. 그리고 필요할 때마다 압축된 핵심 기억을 AI에게 다시 알려주어, 마치 사람처럼 자연스러운 대화의 흐름을 만들어냅니다.

### 핵심 기능 한눈에 보기

- **100% 자동화**: 사용자의 프롬프트, AI의 응답, 도구 사용 기록까지 모든 것을 알아서 캡처합니다. 어떤 추가 작업도 필요 없습니다.
- **지능형 압축**: 자체 개발한 `InstantCompressorV3` 엔진이 대화의 핵심만 남기고 불필요한 부분은 덜어내, 컨텍스트 효율을 극대화하고 비용을 절감합니다.
- **자동 컨텍스트 주입**: AI가 대화 내용을 요약하려 할 때, 시스템이 알아서 압축된 기억을 컨텍스트에 주입하여 AI가 대화의 전체 맥락을 놓치지 않게 합니다.
- **완벽한 호환성**: Claude Code, OpenCode 등 주요 AI 에이전트 환경을 모두 지원하며, 어디서든 일관된 경험을 제공합니다.

### 어떻게 작동하나요?

Conversation Memory V2는 각 AI 플랫폼의 아키텍처에 맞춰 최적화된 방식으로 작동합니다.

1.  **설치**: 사용자가 `memory-factory install` 명령어를 실행합니다.
2.  **감지**: 시스템이 현재 사용 중인 AI 플랫폼(Claude Code 또는 OpenCode)을 자동으로 감지합니다.
3.  **설정**: 플랫폼에 맞는 Hook(Claude Code) 또는 플러그인(OpenCode)을 자동으로 설정합니다.
4.  **실행**: 그 후 모든 대화는 백그라운드에서 자동으로 캡처, 압축, 주입됩니다.

| 플랫폼 | 구현 방식 |
| :--- | :--- |
| **Claude Code** | **Hook + 백그라운드 데몬**: Hook이 대화 조각을 빠르게 캡처하여 큐에 넘기면, 별도의 데몬이 백그라운드에서 압축을 처리합니다. |
| **OpenCode** | **올인원(All-in-One) 플러그인**: 플러그인 내에 내장된 워커가 비동기적으로 모든 작업을 처리하여 더 깔끔하고 효율적입니다. |

### 이런 분들께 필요합니다

- **AI 파워 유저**: AI와 함께 복잡하고 긴 프로젝트를 진행하며 컨텍스트 한계에 부딪히는 분.
- **AI 개발자**: 자신의 애플리케이션에 강력한 장기 기억 능력을 손쉽게 통합하고 싶은 분.
- **LLM 서비스 운영자**: 사용자에게 일관되고 끊김 없는 대화 경험을 제공하며 비용을 절감하고 싶은 분.

### 우리의 비전

Conversation Memory V2는 단순히 대화를 기억하는 것을 넘어, AI 에이전트가 진정한 의미의 **'지속적인 지능(Continuous Intelligence)'**을 갖추도록 하는 첫걸음입니다. 우리는 AI가 과거의 경험으로부터 배우고 성장하여, 모든 사용자와 더 깊고 의미 있는 관계를 맺는 미래를 꿈꿉니다.
