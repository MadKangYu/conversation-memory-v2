## 1. Project Overview

**The Forge** is an autonomous AI coding agent designed to democratize advanced coding capabilities. It leverages the **Gemini 2.0 Flash** model (free tier) to provide a commercial-grade coding experience similar to Claude Code, but without the associated costs. The system is built on a **ReAct (Reasoning + Acting)** architecture, allowing it to plan, execute, and iterate on tasks autonomously.

## 2. Core Architecture

### 2.1 Agent Engine
- **Model**: Google Gemini 2.0 Flash (via OpenRouter or Direct API)
- **Pattern**: ReAct Loop (Think -> Act -> Observe)
- **Context Window**: 10 Million Tokens (supports massive codebases)
- **Memory**: SQLite-based short-term memory + Vector-based long-term retrieval (RAG)

### 2.2 File System Isolation
- **User Space**: Clean workspace, only user files are visible.
- **System Space**: All system configurations, databases, logs, and chrome profiles are isolated in the `.forge/` hidden directory.
- **Git Integration**: `.forge/` is automatically added to `.gitignore` to prevent system file pollution.

### 2.3 Knowledge Management (The Garden)
- **Auto-Wiki**: Automatically digests conversations into structured Markdown documentation in `docs/`.
- **Categories**:
  - `architecture`: System design and structural decisions.
  - `features`: Functional specifications and requirements.
  - `issues`: Bug reports and troubleshooting logs.
  - `decisions`: Architectural Decision Records (ADR).
- **Query Archiving**: All user queries are time-stamped and archived in `docs/history/queries.md` for historical context.
