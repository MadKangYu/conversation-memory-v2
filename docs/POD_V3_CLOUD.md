# Conversation Memory V3: Cloud Edition - 제품 개요서 (POD)

---

## 당신의 AI, 이제 클라우드에서 하나가 되다

**Conversation Memory V3는 로컬에 갇혀 있던 AI의 기억을 클라우드로 해방시킵니다. 회사에서 나누던 대화를 집에서도 이어가고, 수년 전의 기억도 생생하게 떠올리는 진정한 'Second Brain'을 경험하세요.**

---

### 왜 Cloud Edition인가요?

우리는 하루에도 수십 번씩 장소와 기기를 옮겨 다니며 일합니다. 하지만 당신의 AI 에이전트는 각 기기마다 따로따로 기억을 가지고 있어, 매번 새로운 설명을 해야 했습니다. V3는 이 **'기억의 단절'**을 끝내기 위해 탄생했습니다.

| V2 (Local) | V3 (Cloud Edition) |
| :--- | :--- |
| **기기별 독립 기억** | **모든 기기 실시간 동기화** |
| **최근 대화 중심** | **벡터 검색으로 무한한 기억** |
| **CLI 대시보드** | **아름다운 웹 대시보드** |
| **텍스트 요약** | **의미 기반 심층 이해** |

### 핵심 기능: The Trinity of Memory

#### 1. Ubiquitous Sync (어디서나 동기화)
회사 컴퓨터를 끄고 집에 도착해 노트북을 여는 순간, 회사에서 나누던 대화의 맥락이 이미 와 있습니다. Supabase의 강력한 실시간 동기화 기술로 당신의 모든 작업 흐름이 끊김 없이 이어집니다.

#### 2. Semantic Recall (의미 기반 회상)
"그때 그 결제 모듈 에러 어떻게 고쳤지?"라고 물어보세요. 정확한 날짜나 파일명을 몰라도 괜찮습니다. V3는 대화의 **'의미'**를 이해하는 벡터 검색(Vector Search)을 통해, 수만 건의 대화 속에서 당신이 찾는 그 순간을 정확히 찾아냅니다.

#### 3. Visual Insight (시각적 통찰)
검은 터미널 화면은 이제 그만. 직관적인 웹 대시보드에서 당신의 기억 지도를 탐험하세요. 어떤 프로젝트에 시간을 많이 썼는지, 어떤 주제로 대화를 많이 나눴는지 시각적으로 확인할 수 있습니다.

### 기술적 혁신: Hybrid Architecture

V3는 클라우드의 편리함과 로컬의 빠름을 모두 잡았습니다.

- **Offline-First**: 인터넷이 끊겨도 걱정 없습니다. 로컬 DB(SQLite)가 모든 대화를 안전하게 저장하고, 연결되는 즉시 클라우드와 동기화합니다.
- **Secure by Design**: 당신의 기억은 소중하니까요. 모든 데이터는 암호화되어 전송되며, 철저한 인증 시스템으로 보호됩니다.

### 지금 시작하세요

Conversation Memory V3는 단순한 업그레이드가 아닙니다. 당신의 AI 파트너를 **'단순한 챗봇'**에서 **'나를 가장 잘 아는 동료'**로 진화시키는 열쇠입니다.

> **"기억은 클라우드에, 통찰은 당신에게."**
